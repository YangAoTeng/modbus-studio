# 更新记录

## V1.1.4

- 寄存器字典支持自定义排序（上移/下移）。
- 寄存器字典页面新增 Modbus 地址区段说明注释。
- 寄存器字典支持导入/导出（CSV / JSON）。
- 报文日志支持导出（CSV / TXT）。
- 报文日志新增按地址区段筛选（线圈/离散输入/输入寄存器/保持寄存器）。
- Client 与 Server 新增更多波特率选项（1200 ~ 921600）。
- CRC 校验失败时错误消息附带原始 HEX 响应数据，便于排查。

## V1.1.3

- Client 改为寄存器字典驱动读取。
- Server 改为按寄存器字典生成数据区。
- 支持保存 Client 数据、Server 数据、寄存器字典、工程配置和报文日志。
- 修复 Electron 打包后静态资源绝对路径导致白屏的问题。
- 增加开源项目 README、贡献指南、安全策略和 MIT 协议。

## V1.1.0

- 增加 Modbus TCP Client。
- 增加 Modbus TCP Server 和 Modbus RTU Server。
- 增加工程管理、寄存器字典和报文日志视图。
